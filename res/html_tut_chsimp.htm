新手入门

在此您可以找到演示视频的链接和一些简短的指南 (需要连接互联网)，指导您用 xplorer2 来处理一些基本的文件管理任务。如果你访问在线漫游将获得更多的知识, 你也可以浏览博客周刊和产品信息页。

  • 用 xplorer2 图形界面找到熟悉感
    所有的显示面板都可以调整和打开/关闭，从而适合您的浏览习惯
  • 使用功能区界面
    您可以使用功能区工具栏或传统菜单管理 xplorer2 命令. 功能区占用更多空间, 但是可以更直观的组织命令
  • 使用文件夹标签
    用带标签的界面是同时显示许多文件夹的完美解决方案，也是方便文件复制的途径。文件夹组群也可以保存起来
  • 停驻或浮动面板
    公用程序面板，比如文件树和快速查看器能停驻在任何窗口侧边，甚至可以浮动或完全分离
  • 自动隐藏面板
    要获得更多空间而不必隐藏可停驻面板(如目录树)，可切换到自动隐藏模式，使用大头针按钮。
  • 丰富的文件属性
    在详细视图模式，您可以从超过300种信息中选择显示列。加上 [详细信息] 面板和信息栏，您有文件属性的绝对控制权。
  • 浏览您的文件夹
    有许多途径可以快速定位您的文件夹，如使用面包屑结构、驱动器栏和书签(文件收藏夹)
  • 管理书签和启动程序
    迷你面板非常灵活并且可以用来存储文件夹(书签)和您喜欢的程序
  • 只显示您想要的文件
    您可以方便地使用通配符来查看匹配的文件，它具有自动和快速过滤功能。这样您就可以把注意力集中在您需要的项目中
  • 输入时进行选择与过滤
    使用紧靠地址栏的多功能工具进行选择、过滤和查找关键词.
  • 用颜色标记文件及文件夹 新 
    在任何文件夹中用颜色永久标注项目, 使得他们突出且可快速找到
  • 预览 PDF 和 Office文档
    PDF 和其它复杂的文档类型都可以在缩略图视图下预览(第一页)，当然也可以预览纯文本。 您甚至可以精确(activeX)预览而且无需打开相关的程序
  • 检验文件元数据
    详细视图模式支持额外信息列。 对于音乐文件夹您可以选择 ID3 标签列以查看比特率、演唱者等详细资料.
  • 标记或注释照片
    给所有文件类型包括照片、视频设置注释，用以记住他们的相关信息.
  • 直观地修改文件属性
    在文件详情上点击以编辑多个属性. 就像使用扩展表单一样!
  • 基本文件管理
    创建文件夹，复制/粘贴文件和使用鼠标 右键 进行拖放操作
  • 轻松选择
    粘滞选择模式和迷你面板可以帮您轻松选择项目，无需担心遗漏它们
  • 查找文档中的关键词
    xplorer2 有强大的搜索引擎能让您控制微小的细节，例如搜索的 位置 ， 搜索的文件 类型 及其他个性化属性.
  • 查找最近修改的文件
    查找那些在最近修改日期范围内，并且包含关键词的文件。高亮预览匹配关键词的搜索结果
  • 清理您的硬盘以释放空间
    使用文件夹大小统计分析来发现您硬盘中的大文件夹并清理重复文件
  • 文件夹同步基本操作
    双面板视图是比较和同步相似内容文件夹的完美解决方案
  • 分级比较文件夹
    用镜像浏览和滚屏来比较和同步多层级的文件夹
  • 打印文件夹列表
    您在任何面板中看到的内容都可以被导出为文本文件；这样您就可以打印文件夹内容了
  • 自定义工具栏图标
    您可以给任何工具栏包括书签添加任何带图标的命令。在此您也可以选择一个自定义文件夹图标
  • 修改按键绑定
    改变类似 Windows 资源管理器中 F5 键刷新文件夹的键
  • 地址栏中路径自动完成
    您可以用智能的自动完成来保留许多文件夹输入历史记录, 书签和环境变量
  • 给某些文件夹设置自定义视图
    如果您希望您的全部照片都能在缩略图模式下显示，您可以试一试默认视图模式外的其它模式
  • 在群组中安排条目
    支持群组方式，例如，根据名称或修改日期分组，用以对简单的条目排序进行补充。 您可以折叠或展开群组， 甚至定义您自己的分类。
  • 支持在压缩文件夹中运行
    压缩的ZIP 压缩包处理方法与其它文件夹一样。您可以添加/删除文件并预览内容，这一切都无需解压缩文件
  • 作为附件发送到 Hotmail
    通过电子邮件附件发送最方便的办法当然是拖放到 Outlook，不过在 xplorer2 上操作也有一些有用的技巧
  • 拖放到任何地方
    全部显示的环境都是拖放功能的拖动和跳至的标签和文件夹
  • 切换图形界面语言
    xplorer2 被翻译成许多语言，您可以在安装时进行选择
  • 和您的伙伴共享文件
    如果您喜欢，您可以使用文件夹共享选项让您的照片和音乐共享给您的电脑上或本地网络上的其它用户
  • 自定义文件夹图标
    如果您挂上了许多硬盘或者您想要高亮显示某些特定的文件夹, 您可以设置醒目的图标来方便辨别它们
  • 备份或迁移您的设置
    如果您使用不同的电脑，您可以把您的 xplorer2 设置(书签、按键分配等等) 迁移到您的新电脑上
  • 我该如何撤销文件操作？
    xplorer2 没有撤销命令，但是大多数危险的操作(例如意外的拖放操作)都有称手的帮助信息
  • 键盘快捷键
    xplorer2 中几乎任何操作都有键盘快捷键，这里列举了一些"隐藏"的键盘命令

高级主题

  • 用宏自动执行任务  新 
    简单的脚本语言控制鼠标和键盘，帮助您简化经常重复进行的文件管理操作
  • 与窗口布局兼容
    一个窗口风格不一定适合您所有的文件管理工作. 使用窗口布局来切换工具栏和面板排列
  • 拆分与合并大文件
    如果您想要通过电子邮件发送一个大文件，最好先把它拆分为小的部分
  • 永久地粉碎文件
    即便删除的文件也可以被恢复。如需永久地摧毁铭敏感文档，请使用粉碎命令
  • 浏览网络文件夹
    您可以使用 UNC 路径来访问您的本地网络电脑上的资源
  • 解Q UAC 问题
    Windows Vista 及之后的版本会每隔几秒钟弹出 UAC 确认提示，您可以在系y管理员模式下运行 xplorer2 来屏蔽该提示。
  • 启动外部工具并把文件做为参数变量
    您可以在地址栏输入工具名称来启动外部工具 (甚至是MSDOS命令)，也可以指定某些浏览过的文件做为参数变量
  • 复制时的覆盖控制
    xplorer2 的增强型复制引擎可以自动覆盖较旧的文件，某种程度上类似于备份
  • 用 TSVN 控制文件版本
    xplorer2 与流行的 Tortoise SVN 合作，在详细信息视图模式下，把版本系统显示图标集成在状态信息中
  • 我的文件多大阿？
    xplorer2 了解影响文件存储的全部因素，包含扇区大小，压缩过的文件和交替数据流(ADS)
  • 用 ID3 标记重命名 MP3 文件
    块重命名功能可以添加文件元数据中的任何部分到文件名中，例如从 ID3 属性中调取音轨编号
  • 用颜色区分文件名
    您可以依据文件扩展名或相关属性来为您的文件选择颜色
  • 标记文件与文件夹
    只需要一点创意，您就可以使用文件注释和颜色编码规则来标记每个文件(例如，标记为正在处理)
  • 利用类 google 方式搜索关键词
    使用可选的布尔值逻辑操作搜索关键词来搜索您的桌面文件，一如您在互联网上搜索一样
  • 使用文件夹链接重新放置您的文件夹
    如果您的 C:\ 分区装得太满了，您可以移动大文件如Outlook到其他位置，而不会影响程序运行
  • 加密您的文件来保护敏感数据
    加密文件夹来确报您的文档只有您才能看到
  • 带过滤器复制文件
    在复制文件夹时，使用通配符和其他规则来在排除临时文件
  • 删除任意匹配的文件
    您想删除某一文件夹下所有DESKTOP.INI文件而不删除其它文件吗? 可以使用带过滤器的强制删除功能.
  • 恢复意外删除的文件
    如果您已经删除了某些文档并且没有在回收站中找到它们，您仍然可以找回它们，如果您下手够快的话
  • 高级文件管理脚本
    就真正复杂的文件操作而言，xplorer2 没法做到，不过使用 Windows 脚本和现代版本的 DOS 批处理文件可以帮您做到
  • 高级复制/粘贴
    有一些有用的高级操作命令放在编辑 > 选择性粘贴菜单下面，例如只粘贴子文件夹结构，硬链接和文件夹链接

注意：观看视频遇到问题，请安装最新的 flash 插件 or 使用IE播放。 如果视频超出屏幕，请按 Ctrl+减号 慢慢缩小 (或按住 Ctrl 同时滚动鼠标滚轮)。
