Bem-vindo ao xplorer²!
O explorer² se integra perfeitamente à interface gráfica do Windows. Existem muitas extensões de interface gráfica de sistema desenvolvidas por terceiros que melhoram sua vida nos ambientes de trabalho cibernéticos e funcionam tanto com o Explorer quanto com o xplorer². Os downloads sugeridos a seguir podem ser usados gratuitamente se for para uso pessoal.

Prévia de texto e pesquisa no ambiente de trabalho
Filtros de texto criados para servidores de indexação de ambientes gráficos oferecem prévia e pesquisa de texto de muitos tipos de documento, como os do MS Office, PDF, WordPerfect, OpenOffice e outros. Para mais informações e downloads, clique aqui

Controladores de prévia
Se você tem o Windows Vista ou posterior, existem muitos controladores de prévia que funcionam com a guia de prévias Nativa do xplorer². Por exemplo, veja a seguir alguns usados em prévias de formatação de código/linguagem de programação

Controladores de colunas
Adiciona informações extras ao modo de exibição detalhada de pastas/arquivos de acordo os tipos de arquivo, além de mais metadados para serem usados em pesquisas. Veja alguns exemplos:

  • Extensões de MP3 Controladores de colunas de identificadores de MP3/ID3
  • Extensão de Tamanho de pastas (funciona no Windows Vista/7 também, com o xplorer²)

Caso seu Windows seja de 64 bits, não deixe de ver se a versão das extensões escolhidas é para sistemas de 64 bits. Se não conseguir achar uma extensão que não tenha a versão de 64 bits, mas que deseja muito usar, você pode instalar a versão de 32 bits do xplorer².
