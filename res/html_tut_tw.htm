入門

在此您可以找到展示視訊與一些簡短指南的連結 (需要網際網路連線) ，使您在 xplorer² 進行基本的檔案管理工作。 如果您取得線上指南，瀏覽週記部落格與產品資訊頁面您將找到更多有用的資訊。

  • 取得熟悉的 xplorer² GUI
    可以調整所有顯示窗格的大小與開啟/關閉以滿足您的瀏覽習慣
  • 使用功能區 UI
    您可以有功能區工具列或傳統功能表以管理 xplorer² 命令。 功能區需要多一點空間，但更直覺地組織管理命令
  • 使用資料夾索引標籤工作
    索引標籤式介面是同時顯示許多資料夾和相互複製檔案的想法。 資料夾群組也能儲存
  • 停駐或浮動面板
    公用程式窗格，比如樹狀和快速檢視能停駐在任何視窗側邊，或甚至浮動和完整分離
  • 自動隱藏面板
    為了要求更多空間而不隱藏可停駐面板，如樹狀面板，使用他們的釘選按鈕切換為自動隱藏模式
  • 豐富檔案屬性
    在詳細資料檢視模式，您可以選擇顯示超過 300 種資訊。 加上 [詳細資料] 窗格和資訊列，您有檔案屬性的絕對控制權。
  • 瀏覽您的資料夾
    有許多方法快速查詢資料夾，使用階層連結、磁碟機列和書籤 (最愛資料夾)
  • 管理書籤並啟動程式
    迷你虛擬資料夾窗格是非常靈活並能用來儲放資料夾 (書籤) 和最愛的程式
  • 只顯示您要的檔案
    您可以方便地使用自動篩選器和快速篩選器查看與萬用字元相符的檔案。 強制注意您所需要的資料夾部分。
  • 輸入時選取並篩選
    使用網址列旁多用途工具來選取、篩選及搜尋關鍵字。
  • 色彩標籤檔案和資料夾
    在任何資料夾中永久以色彩標記項目，使其脫穎而出和快速找到它們
  • 預覽 PDF 與 office 文件
    PDF 與其它複雜文件類型可以在縮圖預覽 (第一頁)，或以純文字模式。 您甚至可以精確 (activeX) 預覽而不開啟關聯的程式
  • 檢閱檔案中繼資料
    詳細資料檢視模式支援額外資訊欄位。 在音樂資料夾您可以選取 ID3 標籤欄位以查看詳細資料，比如位元率、演唱者等。
  • 標籤或註解照片
    設定所有檔案類型的註解，包含照片和視訊，以記住它們的相關資訊。
  • 直覺變更檔案內容
    按一下檔案詳細資料編輯許多內容，如您在資料夾所見。 就如同使用試算表!
  • 基本檔案管理
    建立資料夾、複製/貼上檔案和使用滑鼠右鍵拖放
  • 輕鬆建立選取範圍
    相黏選取模式和迷你剪輯資料窗格協助您選取項目而不用擔心失去它們
  • 尋找文件中關鍵字
    xplorer² 有強大的搜尋引擎讓您能控制微小細節，比如搜尋的位置，尋找文件的類型和其個別內容。
  • 尋找最近修改的檔案
    尋找在過去幾天修改的檔案和包含一些文字 (關鍵字)。 預覽搜尋結果醒目提示相符的關鍵字
  • 清理您的硬碟以釋放空間
    使用資料夾大小統計探索您硬碟中的大資料夾並清理重複檔案。
  • 基本資料夾同步
    窗重窗格檢視使用較小的內容比較和同步資料夾是完美的
  • 比較資料夾層次結構
    以鏡像瀏覽和捲動比較及同步深層資料夾階層
  • 列印資料夾清單
    您可以匯出任何看到的窗格為文字; 一種可能用途是列印資料夾內容
  • 自訂化工具列圖示
    您可以加入任何命令加上圖示在任何的工具列，包含書籤。 在此您也可以選擇自訂資料夾圖示
  • 變更按鍵繫結
    變更 F5 鍵來重新整理資料夾如同在 Windows 檔案總管
  • 在位址列自動完成路徑
    您可以使用智慧自動完成從資料夾歷程記錄和環境變數儲存許多輸入。
  • 設定某些資料夾的自訂檢視
    如果您要所有的圖片資料夾以縮圖模式顯示，您可以中斷預設檢視模式
  • 以群組安排項目
    分組補充簡單項目排序，例如: 依名稱或修改日期。 您可以摺疊和展開群組，甚至定義您自己定義的分類。
  • 在 Zip 資料夾中工作
    已壓縮 ZIP 壓縮檔處理任何其他資料夾。 您可以新增/移除檔案和預覽內容不需解壓縮檔案
  • 附加檔案至 Hotmail
    透過 Internet 電子郵件傳送附件不像拖放到 Outlook 那麼容易，但 xplorer² 有一些有用的技巧
  • 隨處拖放
    所有顯示的元素皆有拖放感知功能，拖曳捲動和跳躍載入的索引標籤和資料夾
  • 變更 GUI 語言
    xplorer² 已翻譯為許多語言，您可以在安裝期間選取
  • 與其他人共用檔案
    如果您要使照片和音樂與在您電腦或區域網路的其他使用者可用，使用資料夾共用選項
  • 自訂資料夾圖示
    如果您有許多附加的硬碟或如果您要醒目提示某些特殊資料夾，設定不同的圖示來分開對待。
  • 備份或傳輸您的設定
    如果您變更電腦，您可以取得您的 xplorer² 設定 (書籤、鍵盤指派等) 到您的新 PC
  • 我如何能復原操作?
    xplorer² 沒有復原命令，但大多數危險的操作 (例如 意外拖放) 協助就在眼前
  • 鍵盤快速鍵
    幾乎所有在 xplorer² 的操作有鍵盤快速鍵，這裡是一些「隱藏」鍵盤命令

進階主題

  • 使用巨集自動工作  新增 
    簡單指令碼語言控制滑鼠和鍵盤，有助於您經常重複的檔案管理操作。
  • 與視窗配置工作
    一個視窗風格不適合您的所有檔案管理工作。 使用視窗配置切換工具列和安排窗格
  • 分割與結合大檔案
    如果您要以電子郵件傳送大檔案，最好是先剪成較小的區塊
  • 永久銷燬檔案
    即使刪除檔案也能還原。 若要永久刪除敏感文件，使用粉碎命令
  • 瀏覽網路資料夾
    您可以使用 UNC 路徑存取在您本機網路電腦的資源
  • 解決 UAC
    Windows Vista 和之後每隔幾秒彈出 UAC 確認提示，您可以在系統管理模式執行 xplorer² 使其沉默。
  • 啟動外部工具並傳送檔案為引數
    您可以啟動外部工具 (甚至 MSDOS 命令)，在位址列輸入名稱並選用的傳送某些瀏覽的檔案作為引數
  • 複製時覆寫控制
    xplorer² 有增強型複製引擎能自動只覆寫較新的檔案，此行為就像是備份。
  • 使用 TSVN 控制檔案版本
    xplorer² 與流行的 Tortoise SVN 版本系統合作，顯示圖示重疊並整合狀態資訊在詳細資料檢視模式中
  • 我的檔案多大?
    xplorer² 知道有關所有影響檔案存放裝置的係數，包含磁柱大小、壓縮的檔案和和甚至替代資料串流 (ADS)
  • 使用 ID3 標籤重新命名 MP3 檔案
    眾人的重新命名工人能在檔案名稱中加入任何的檔案中繼資料部分，例如 ID3 屬性的音軌數
  • 色碼檔案名稱
    您可以根據附檔名或複雜屬性選擇檔案的色彩
  • 搜尋關鍵字類似 Google
    使用可選的布林邏輯運算字元以搜尋關鍵字來搜尋您的桌面檔案，就像是搜尋網際網路
  • 使用資料夾聯合以重新安排您的資料夾
    如果您的 C:\ 分割區變得太擁擠，您可以移動大資料夾到其它位置而不中斷程式，比如 Outlook
  • 加密您的檔案以保護敏感資料
    加密資料夾確認只有您可以開啟您的文件
  • 使用篩選器複製檔案
    使用萬用字元或其它規則在複製資料夾時排除暫存檔案
  • 只刪除相符的檔案
    您要刪除資料夾下所有的 DESKTOP.INI 檔案嗎? 使用含篩選的強大刪除。
  • 復原意外刪除的檔案
    如果您已刪除一些文件且無法在資源回收筒中找到，如果您迅速採取行動，您仍然可以將它取回
  • 進階檔案管理指令碼
    對於真正複雜 xplorer² 無法處理的檔案操作，使用 Windows scripting host，DOS 批次檔案的現代版
  • 進階複製/貼上
    某些有用的命令在 [編輯特殊貼上] 功能表下作為進階操作，例如 只貼上子資料夾結構、硬式連結和資料夾聯合

備註: 如果您有任何視訊的問題，請安裝最新的 Flash 外掛程式 或使用 Internet Explorer。 如果視訊解析度似乎關閉，請按 Ctrl + 減號鍵稍微 縮小 (或在使用滑鼠滾輪時按住 Ctrl 鍵)。

 

字型太小無法閱讀嗎?

您可以按下 Ctrl + 加號鍵，或滾動滑鼠滾輪時按住 Ctrl 鍵使文字變大。
