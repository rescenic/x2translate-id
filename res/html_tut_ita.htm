Da dove partire

In questo documento troverete una serie di link a video dimostrativi e ad alcuni brevi tutorial (collegamento ad internet richiesto) che vi introdurranno alla gestione di base dei file in xplorer. Troverete ulteriori informazioni utili seguendo il tour online, seguendo il blog settimanale e la pagina di informazioni sul prodotto.

  • Familiarizzare con la GUI di xplorer
    Tutti i pannelli visualizzabili possono essere ridimensionati e nascosti per venire incontro alle vostre preferenze di navigazione
  • Lavorare con le tab delle cartelle
    L'interfaccia a tab è ideale per mostrare diverse cartelle contemporaneamente e copiare file dall'una all'altra. Si possono persino salvare gruppi di cartelle.
  • Pannelli agganciabili o liberi
    I pannelli di utilità, come l'albero cartelle e il visualizzatore anteprime, possono essere agganciati a qualunque bordo finestra, oppure lasciati liberi di muoversi in maniera autonoma
  • Numerosi attributi file  NUOVO 
    Nella modalit à dettagli potete scegliere tra più di 300 differenti informazioni da visualizzare. Insieme al pannello dettagli e alle barre informazioni avrete controllo assoluto sugli attributi dei file.
  • Navigare tra le cartelle
    Ci sono diversi modi di navigare velocemente tra le vostre cartelle utilizzando i breadcrumbs, la barra dei drive e i segnalibri (cartelle preferite)
  • Gestire segnalibri ed eseguire programmi
    Il mini-raccoglitore è molto flessibile e può essere usato per memorizzare cartelle (tramite segnalibri) e programmi preferiti.
  • Mostrare solo i file che desiderate
    Potete visualizzare con semplicit à file che soddisfino wildcards tramite il filtro automatico e i filtri veloci. Concentrate la vostra attenzione sulla parte delle cartelle che vi interessa.
  • Visualizzare anteprime di file PDF e altri documenti
    I file PDF e altri tipi di documento complesso possono essere visualizzati in modalità anteprima (prima pagina), oppure in modalità solo testo. Potete anche avere una anteprima completa (tramite activeX) senza dover aprire il programma associato.
  • Controllare metadati dei file
    La visualizzazione in modalità Dettagli supporta colonne di informazione aggiuntive. Per le cartelle di musica potete selezionare colonne ID3 tag per avere informazioni riguardo bitrate, artisti etc.
  • Commentare immagini
    Impostate commenti per ogni tipo di file, incluse immagini e video, per ricordarvi di cosa si tratta.
  • Gestire file
    Create cartelle, copiate/incollate file e fate drag-drop con il tasto destro del mouse.
  • Creare le selezioni con semplicità
    La modalità di selezione permanente ed il mini-raccoglitore vi aiutano a ragruppare elementi senza preoccuparvi di perdere la selezione
  • Trovare parole chiave nei documenti
    xplorer possiede un potente motore di ricerca in cui potete controllare in dettaglio dove cercare, in quali tipi di documento, e con quali proprietà individuali.
  • Trovare file modificati di recente
    Trovate file modificati nell'ultimo giorno e contenenti il testo desiderato (parole chiave). Visualizzate un'anteprima dei risultati mettendo in risalto le parole chiave trovate.
  • Pulire il disco fisso per liberare spazio
    Utilizzate le statistiche di dimensione delle cartelle per individuare le cartelle più grandi sul vostro disco fisso e fare pulizia tra i file duplicati.
  • Sincronizzare cartelle
    La visualizzazione a doppio pannello è ideale per confrontare e sincronizzare cartelle con contenuti simili.
  • Confrontare di gerarchie di cartelle
    Confrontate e sincronizzate in profondità gerarchie di cartelle tramite scorrimento e navigazione sincronizzati.
  • Stampare i contenuti di una cartella
    Tutto ciò che è visibile in qualunque pannello può essere esportato come testo; uno degli usi possibili è la stampa dei contenuti di una cartella.
  • Personalizzare le icone della toolbar
    Potete aggiungere un qualunque comando tramite un'icona in qualunque barra degli strumenti, inclusi i segnalibri, dove potete anche selezionare un'icona personalizzata per le cartelle.
  • Modificare l'associazione dei tasti
    Modificate il comportamento del tasto F5 per rinfrescare le cartelle come in windows explorer
  • Completamento automatico dei percorsi nella barra degli indirizzi
    Potete ridurre l'uso dei tasti mediante il completamento automatico dalla cronologia delle cartelle, dai segnalibri e dalle variabili d'ambiente.
  • Impostare visualizzazioni personalizzate per cartelle specifiche
    Se volete visualizzare tutte le vostre cartelle di immagini in modalità anteprima potete modificare la modalità di visualizzazione standard.
  • Raggruppare gli elementi  NUOVO 
    Il raggruppamento affianca l'ordinamento semplice degli elementi ad. es. per nome o data di modifica. Potete chiudere o espandere i gruppi, e persino definire delle categorie personalizzate per il raggruppamento.
  • Lavorare con cartelle compresse
    Archivi ZIP compressi sono gestiti come una qualunque altra cartella. Potete aggiungere, rimuovere file e vedere in anteprima i contenuti senza estrarre i file.
  • Allegare file ad Hotmail
    Mandare allegati via posta elettronica non è semplice come fare drag-drop verso Outlook, ma xplorer supporta alcuni utili scorciatoie
  • Drag-drop dappertutto
    Tutti gli elementi visualizzati sono abilitati al drag-drop, supportano lo scorrimento durante il trascinamento e le tab e le cartelle sono sensibili al drop
  • Modificare la lingua della GUI
    xplorer è tradotto in varie lingue, potete selezionare la vostra durante l'installazione.
  • Condividere file con gli altri
    Se volete rendere disponibili immagini o musica anche ad altri utenti sul vostro PC o sulla vostra rete, usate la funzionalità di condivisione delle cartelle.
  • Personalizzare le icone delle cartelle
    Se avete molte unità a disco collegate, o desiderate distinguere cartelle speciali, impostate un'icona distintiva per ciascuna.
  • Fare backup o trasferire le vostre impostazioni
    Se state cambiando computer, potete esportare le vostre preferenze di xplorer (segnalibri, scorciatoie tasti ecc.) verso il nuovo PC
  • Come annullare operazioni sui file?
    xplorer non possiede un comando di Undo ma le operazioni più pericolose (ad es. drag-drop accidentale) c'e' un aiuto disponibile.
  • Scorciatoie da tastiera
    Praticamente tutto in xplorer ha un acceleratore di tastiera, qui troverete alcuni comandi di tastiera "segreti"

Argomenti avanzati

  • Dividere ed unire file di grosse dimensioni
    Se volete spedire un file di grandi dimensioni tramite email potrebbe essere meglio suddividerlo in parti più piccole prima.
  • Rimuover in maniera permanente i file
    Anche i file eliminati possono essere recuperati. Per rimuovere permanentemente documenti sensibili utilizzate il comando Distruggi
  • Navigare cartelle di rete
    Potete accedere alle risorse della vostra rete locale tramite percorsi UNC
  • Aggirare l'UAC
    Windows vista e successivi visualizzano continuamente finestre di conferma per l'UAC, ma potete evitarlo eseguendo xplorer in modalità amministratore.
  • Eseguire tool esterni e passare nomi di file come argomenti
    Potete eseguire tool esterni (anche comandi MSDOS) digitando il loro nome nella barra degli indirizzi ed eventualmente passare il nome dei file visualizzati come argomento.
  • Controllare la sostituzione dei file durante la copia
    xplorer possiede un motore di copia robusta che può automaticamente sovrascrivere solo i fle più recenti, funzionando così come un sistema di backup.
  • Controllare il versionamento dei file tramite TSVN
    xplorer collabora con il popolare strumento di versionamento file Tortoise SVN mostrando overlay di icona e integrando le informazioni di stato nella visualizzazione dettagli.
  • Quanto sono grandi i vostri file?
    xplorer conosce tutti i fattori che influenzano la memorizzazione dei file, inclusi la dimensione dei settori, la compressione e anche gli ADS (alternate data streams).
  • Rinominare file MP3 utilizzando i loro tag ID3
    La funzionalità di rinominazione di gruppo è in grado di aggiungere qualunque parte dei metadati dei file al nome del file, ad es. il numero di traccia contenuto negli attributi ID3
  • Nomi di file colorati
    Potete specificare il colore dei vostri file sulla base della loro estensione o dei loro attributi.
  • Ricercare parole chiave in stile google
    Utilizzate gli operatori logici booleani con le parole chiave per cercare tra i file del desktop come fareste su internet.
  • Usare le giunzioni di cartella per riorganizzare le cartelle
    Se la vostra partizione C:\ sta diventando troppo affollata, potete spostare le cartelle più grandi in posizioni diverse senza interferire con il funzionamento di programmi come Outlook
  • Cifrare i file per proteggere informazioni sensibili
    Cifrare una cartella vi garantisce che i suoi contenuti saranno visibili solo da voi.
  • Copiare file con filtri
    Utilizzate wildcards e altre regole per escludere file temporanei quando copiate una cartella.
  • Recuperare file eliminati accidentalmente
    Se avete eliminato qualche documento e non riuscite a trovarlo nel Cestino, potete ancora recuperarlo agendo tempestivamente.
  • Script avanzati di gestione file
    Per operazioni su file complesse non supportate da xplorer, utilizzate il windows scripting host, la versione moderna dei file batch del DOS
  • Copia e incolla avanzati
    Ci sono alcuni comandi utili nel menu Modifica > Incolla speciale per operazioni avanzate, ad es. incolla la struttura delle cartelle, hard links e giunzioni di cartella

NOTA: alcune delle precedenti caratteristiche sono disponibili solo per xplorer professional, ad es. le voci di menù indicate con [P]. La versione professional dispone di una trial gratuita di 21 giorni. Se avete problemi con qualche video, per favore installate l'ultima versione del plugin flash.
